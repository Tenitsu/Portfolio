Finding Alice

An immersive environment project with an emphasis on user experience design. The idea is to create an environment where the user will suspend their disbelief and enjoy a rich experience without feeling lost.

Project Half Attention Please - a User experience/ User interface project

Project Half-attention please is a User experience/ User interface project where we tackle the problem of student distraction in lecture halls due to electronics. Our aim is to understand how students use technology in lectures and how we can use that to design an application which would improve the learning experience but doesn’t discourage the use of technology.

Da Taipei Business Card Layout Design

A project I did for a food stand in Taiwan. The challenge was to work with graphic design in a different language but still effectively communicating important information.

Denebola The Leo

An original character vector illustration created using Adobe Illustrator. Given the theme of Leo, I designed and illustrated this piece to give off a relaxed atmosphere.

Magazine Redesign

The challenge was to redesign an existing magazine in order to increase its appeal against competition. By combining my knowledge of layout design and typography, I managed to craft a more suiting appeal for the actual content it’s presenting.

Exploring Layout

This was a project done to explore layout design. The emphasis is specifically paid to the alignment of text to photos and the effective use of white space.
