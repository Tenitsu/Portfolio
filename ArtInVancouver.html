Project objective

This was a web interface exploration project with the emphasis on interactive video. As the user click on each person, a video will appear with a segment of the documentary we filmed. We decided to make it a one pager and rely on horizontal scrolling to make the user feel as if they are walking through the art gallery.

Process

We started off with only three constraints in mind. First of all, it has to allow us to show our documentary in sections. Secondly, it needs to be unique and draw the user in. Thirdly, it needs to be usable. We brainstormed a lot of ideas and sketched potential situations in which the user might use the website. In conclusion, we went with a horizontal one page scrolling idea and we sketched again. After the idea is solidified, I began creating the illustration in illustrator and Photoshop. Once that is completed, I used Dreamweaver to code and style the website.

More Detail about Project
