Project objective

To create a portfolio to specifically showcase my Unity works.

Process

I was asked to create a portfolio that showcases the work I’ve done with Unity3D and that was my starting point. I started by listing out what exactly is it that I need from this portfolio and using that information to plan out my information architecture. As the keyword is “showcase”, I decided that the best approach is to show screenshot of my work on the home page and provide easy to see navigation at the top right hand corner. Using photoshop, I created a mockup to determine how I should layout everything and then I head over to Dreamweaver to code it.

More Detail about Project

The emphasis of this web design project was usability and information architecture and thus Dreamweaver was used to build the site.
