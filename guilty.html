Project Objective

To create an interactive multi linear story with emphasis on usability, user experience and interface design.

Process

The project started off with a brainstorming session to come up with a story. We made use of flowcharts to work out how each node is connected to each other and how many paths there are. After a story is decided, we created wireframes to quickly test how well our idea works so that we can move onto programming. Lastly, I used Action Script 3.0 to program the interaction and complete the project.

More Detail about Project

The project is primarily a multi linear story. In other words, we are concerned with removing elements that may distract the user from the story to create a suspension of disbelief effect. However, we are also concerned about usability and user experience.
